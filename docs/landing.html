DeepJ

A lightweight Java library for building and experimenting with Transformer-based neural networks.

☕ Pure Java 🔧 Zero Dependencies ⚡ Metal GPU
Get Started ⭐ GitHub 📚 Javadoc
Getting started

Installation

Add the GitHub Packages repository and dependency to your pom.xml.

<dependency>
    <groupId>io.github.kirstenali</groupId>
    <artifactId>deepj</artifactId>
    <version>0.4.0-alpha</version>
</dependency>
Why DeepJ

Features

Everything you need to build, train, and deploy Transformer models — in pure Java.

📐

Tensor Engine

2-D tensors with 60+ operations. Pluggable backends — swap between CPU and Metal GPU with one line.

🧱

Layer Primitives

Linear, FNN (MLP), LayerNorm, Multi-Head Self-Attention, TransformerBlock — all with forward & backward.

🤖

GPT Model

Decoder-only GPT with token & positional embeddings, configurable depth, and autoregressive text generation.

⚡

Metal GPU Backend

Lazy, GPU-resident execution on Apple Silicon. Ops record into a command stream and flush in one batch — up to 1700× speedup.

🏋️

Training Loop

Pluggable Trainer with EMA loss, early stopping, checkpointing hooks. Factories for supervised and causal LM training.

🔤

Tokenizers

ByteTokenizer for quick experiments, full BPE pipeline for subword tokenization. Both implement the Tokenizer interface.

🧮

AdamW Optimizer

Per-parameter state, bias correction, weight decay. Works with any model exposing parameters().

💾

Binary Persistence

Save & load model weights as compact binary files. No external format dependencies.

💬

Chat UI

Optional JavaFX chat interface. Load a trained model, type a prompt, and generate text — model-agnostic via ChatService.

Under the hood

Architecture

DeepJ is organised into focused, single-responsibility packages.

layers.transformerTransformerBlock, multi-head attention, layer norm

transformer.embeddingsEmbedding (token lookup), PositionalEmbedding
activationsGELU, ReLU, Sigmoid, Tanh, Softmax
lossMSELoss, CrossEntropyLoss
optimisersAdamW, Parameter
trainingTrainer, SupervisedTraining, CausalLMTraining
models.gptGPTModel, GPTConfig, TextGenerator
tokenizersByteTokenizer, BPE pipeline

chatuiBaseChatApp, ChatService

Key Interfaces

Trainable — exposes parameters() + zeroGrad() └── Layer — forward(Tensor) / backward(Tensor) Persistable — save(Path) / load(Path) via ModelSerializer TensorBackend — all tensor ops, swappable (CpuBackend / MetalBackend) LossFunction — loss() + gradient() ParameterOptimizer — step(List<Parameter>) Tokenizer — encode(String) / decode(int[]) / vocabSize()
Apple Silicon

Metal GPU Backend

Lazy, GPU-resident execution. Tensors stay on the GPU between operations — data only transfers when you need it.

Enable in one line

MetalBackend metal = new MetalBackend();
Tensor.setBackend(metal);

Every supported Tensor operation now routes through Metal. CPU-only ops materialise automatically as needed.

How it works

CPU GPU │ │ │ a = random(512,512) ──▶ buffer A │ b = random(512,512) ──▶ buffer B │ │ │ c = a.matmul(b) record op │ d = c.relu() record op │ e = d.softmaxRows() record op │ │ │ e.materialize() flush all │ ◀── buffer E │ read e.data[][] │

No round-trips

Intermediate results never leave the GPU

Batched execution

All recorded ops flush in a single GPU command buffer

Staleness tracking

Each tensor knows whether its CPU or GPU copy is current

Examples

Classic MLP (FNN)

FNN mlp = new FNN(
        3,                       // inputSize
        new int[]{16, 16},       // hiddenSizes
        3,                       // outputSize
        GELU::new,               // hiddenActivation
        null,                    // outputActivation
        rnd
);

Trainer trainer = SupervisedTraining.trainer(
        mlp,
        new MSELoss(),
        AdamW.defaultAdamW(3e-3),
        x, y, 123L
);

trainer.train(
        3000,  // maxSteps         – stop after 3 000 gradient updates
        3,     // batchSize        – rows sampled per step
        200,   // logEvery         – print loss every 200 steps
        0.98,  // emaBeta          – smoothing factor for moving-average loss
        1e-6   // targetEmaLoss    – early-stop when smoothed loss falls below this
);

GPT Training + Generation

Tensor.setBackend(new MetalBackend());

GPTConfig cfg = new GPTConfig(
        tok.vocabSize(),
        256,  // maxSeqLen
        512,  // dModel
        4,    // nHeads
        5,    // nLayers
        1025  // dFF
);

GPTModel model = new GPTModel(cfg, 42);
Trainer trainer = CausalLMTraining.trainer(model, ds, 1e-4);

trainer.train(10_000_000, 2, 1, 0.98, 0.01, 25, checkpointHook);

// ── Inference ──
String out = TextGenerator.generate(
        model, tok, cfg,
        "Bob Marley was ",
        200,     // maxNewTokens
        0.1,     // temperature – low = deterministic
        20,      // topK – sample from top 20 tokens
        1234L    // seed
);

Transformer Builder

TransformerStack stack = new TransformerBuilder()
        .dModel(128)              // model dimension
        .nHeads(4)               // attention heads
        .dFF(512)                // feed-forward inner dimension
        .nLayers(4)              // number of decoder blocks
        .ffnActivation(GELU::new)
        .seed(42)
        .build();
